# 🏥 MediLink - Healthcare Website

A modern and responsive Healthcare Website built using React.js and Vite, designed to provide an intuitive experience for patients looking for medical services.

## 🚀 Live Demo

🔗 https://medilinkin.vercel.app

## ✨ Features

- Responsive UI
- Doctor Listing
- Appointment Booking Interface
- React Router Navigation
- Modern Healthcare Landing Page

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router DOM

## 📦 Installation

```bash
npm install
npm run dev
