## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/aftabshekh/medilink-healthcare-app.git
```

Move into project

```bash
cd medilink
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

---

## ☁️ Deployment

The application is deployed using **Amazon S3 Static Website Hosting**.

Deployment Steps:

- Build React Application
- Upload Production Build to Amazon S3
- Configure Bucket Policy
- Enable Static Website Hosting
